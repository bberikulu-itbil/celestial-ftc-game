📖 Руководство

1

Выберите детали

Откройте категории в левом сайдбаре: Моторы, Колёса, Контроллеры, Батареи, Сенсоры. Каждая категория содержит реальные FTC компоненты от REV Robotics, GoBilda и Swift.

2

Перетащите детали

Используйте drag-and-drop для перемещения деталей из сайдбара на слоты робота. Система автоматически проверит совместимость деталей со слотами.

3

Выполните цели уровня

Установите 4 мотора, 4 колеса, подключите Control Hub и батарею. Следите за прогрессом выполнения целей в нижней панели.

4

Протестируйте робота

После выполнения всех целей нажмите кнопку "Тест" для проверки робота. При успешном тесте станет доступен переход на следующий уровень.

5

Сохраните прогресс

Используйте кнопку "Сохранить" для сохранения вашей сборки. Прогресс сохраняется в локальном хранилище браузера.

🌍

Смена языка

Выберите удобный язык интерфейса для игры. Поддерживаются русский, казахский и английский языки.
