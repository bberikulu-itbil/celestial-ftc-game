📖 Руководство по сборке робота FTC

🏗️ Выбор подходящей базы

💡 Совет: Начните с REV Base Plate
REV Base Plate обеспечивает прочную основу для большинства конструкций. Размер 18" x 18" и вес 5.2кг делают его идеальным выбором для начинающих команд.
🔧 Альтернатива: GoBILDA Chassis Kit
GoBILDA Chassis Kit предлагает лёгкий вес (4.8кг) и модульность. Размер 16" x 16" подходит для компактных конструкций.
⭐ Рекомендация
Для лучшей совместимости с другими компонентами выбирайте REV Base Plate. Он обеспечивает оптимальное соотношение прочности и веса для соревнований FTC.

⚙️ Моторы и контроллеры

🎯 Оптимальная конфигурация
Используйте 4 мотора REV Core Hex для достижения максимальной мощности. Каждый мотор обеспечивает 72Вт при весе 450г.
📱 Контрольные хабы
REV Control Hub - основной контроллер с WiFi и Bluetooth. REV Expansion Hub - дополнительный для подключения до 8 моторов. Обязательно используйте оба для полной функциональности.

🛞️ Колёса и шасси

🏎️ Типы колёс
REV Traction Wheel - для максимального сцепления с полем.
REV Omni Wheel - для плавного движения в любом направлении.
REV Mecanum Wheel - для омни-движения и манёвренности.
⚖️ Баланс веса
Старайтесь сохранять баланс между мощностью и весом. Оптимальный вес робота для FTC: 12-15кг.

🔋 Питание и электроника

🔋 Батарея
REV 12V Battery обеспечивает 4000mAh ёмкости. Всегда имейте запасную батарею для соревнований.
📊 Оптимизация характеристик
• Скорость: 2.5-3.0 м/с для манёвренности
• Точность: 70-80% для точных задач
• Надёжность: 80-90% для долгой работы

🎯 Стратегии сборки

📋 Порядок сборки
1. Установите базу
2. Добавьте моторы и колёса
3. Подключите контрольные хабы
4. Установите батарею
5. Добавьте функциональные компоненты
🔧 Тестирование
После сборки обязательно протестируйте все системы: движение, захват, управление. Проверьте надёжность соединений.

🏆 Соревновательные советы

🏆 Правила FTC
• Максимальный размер робота: 45.7cm x 45.7cm x 45.7cm
• Максимальный вес: 16.8кг
• Используйте только сертифицированные компоненты
🎯 Эффективные практики
• Регулярно проверяйте соединения
• Имейте набор инструментов для ремонта
• Тренируйте управление роботом
• Изучайте правила соревнований
